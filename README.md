# alphacamp-express-restaurant-list
Alphacamp 【指標作業】打造餐廳清單

## 「餐廳清單」的基本功能

* 使用者可以在首頁看到所有餐廳與它們的簡單資料
    1. 餐廳照片
    2. 餐廳名稱
    3. 餐廳分類
    4. 餐廳評分

* 使用者可以再點進去看餐廳的詳細資訊
    1. 類別
    2. 地址
    3. 電話
    4. 描述
    5. 圖片

* 使用者可以透過搜尋餐廳名稱來找到特定的餐廳
* 使用者可以透過搜尋餐廳類別來找到特定的餐廳
